# PM2008 Tools Preview

用于修改 PM2008 键位映射及部分其他设置的辅助工具。

> 仅适用于截止 2026/7/5 的版本

## 功能简介

### 键位映射

程序通过临时修改游戏读取键盘输入的过程实现键位映射，不会修改游戏文件。

启动游戏时，程序会：

1. 以暂停状态启动 launcher；
2. 在目标进程中挂接 GetRawInputData；
3. 恢复程序运行；
4. 在游戏读取 Raw Input 键盘数据时，将指定按键直接替换为新的按键

所有修改仅存在于游戏运行期间。退出游戏后，相关内存修改会自动消失。键位映射功能需要对启动后的子进程进行内存写入，部分杀毒软件或安全软件可能会阻止程序运行。

## 系统要求

* Windows
* .NET Framework 4.7.2 或更高版本

## 使用方法

1. 运行 config.exe；
2. 打开“其他设置”页面并选择游戏目录，选择后会立即保存，程序目录下会自动创建 config.json 配置文件；
3. 打开“输入映射”页面，勾选“启动键位修改注入”；
4. 双击“映射1”或“映射2”设置按键。每个游戏控制项最多可以设置两个实体按键，按 Backspace 键可以清空当前映射。

### 重置所有映射

重置所有映射会将“映射1”恢复为默认键位，并清空“映射2”以及所有映射延迟设置。

## 其他设置

程序还提供部分与音频和仿真相关的实验性设置。由于游戏本体及相关功能仍处于测试阶段，这些设置不保证在所有环境下有效，也不保证能够获得预期效果。
